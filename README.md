# Comandos do terminal para a criação e rodar a api


O comando a seguir cria automaticamente um arquivo chamado package.json (RG do projeto)

```
npm init -y 
```

O próximo comando instala o framework Express no projeto

```
npm install express
```

O último comando executa seu arquivo JavaScript com Node.js e mantém atualizado

```
node --watch server.js 
```
